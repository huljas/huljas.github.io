Play framework modules

I'm a fan of Play Framework and have created some modules for it.

  • @messages

    Localization UI which finds the localization keys from your sources. Supports generated keys and makes it easy to remove obsolete keys.
    Sources Example

  • carbonate

    Data migrations tool that user Hibernate schema update to generate the SQL from your model.
    Sources Example

  • antify

    Ant build support that allows you to run your play application with ant. Mainly intended to work with CI like Jenkins.
